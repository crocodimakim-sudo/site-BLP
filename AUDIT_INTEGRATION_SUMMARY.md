## КРИТИЧНЫЕ МОМЕНТЫ ДЛЯ ПОЛЬЗОВАТЕЛЯ

### 🔴 БЕЗОПАСНОСТЬ (MUST FIX)
Если не исправить:
- ❌ Форму может взломать с помощью CSRF (отправить с другого сайта)
- ❌ XSS: если в проектах будут специальные символы → атака через innerHTML
- ❌ Email-injection: можно вставить `\r\nBcc:` и отправлять письма на другие адреса
- ❌ Боты забьют форму спамом (нет rate limiting)
- ❌ Если mail() не работает → заявки теряются (нет логирования)

### 🔴 SEO (QUICK WIN)
- `catalog.php` БЕЗ h1 → не ранжируется
- Главная и каталог БЕЗ meta description → нет сниппета в поиске
- 0 Schema.org на всех сайтах конкурентов → **ВЫ МОЖЕТЕ БЫТЬ ПЕРВЫМИ** с rich snippets!

### 🔴 CSS АРХИТЕКТУРА (TECHNICAL DEBT)
- 113 !important блокируют любое переопределение → невозможно кастомизировать
- 15 breakpoints → конфликты на мобильных
- Каждый page-CSS дублирует всё → +50% лишнего кода
