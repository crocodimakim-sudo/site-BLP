## 2. Анализ соответствия контента поисковому интенту

### index.php — Главная страница

**Целевые запросы:** фиброцементные панели, вентилируемые фасады, фасадные плиты НГ, негорючие фасадные панели

**Проблемы соответствия SERP-интенту:**
- H1 содержит ключевые слова, но не включает уточнение «навесные» (НВФ) — это важный поисковый термин
- Subtitle «Архитектурные панели окрашенные в массе» — специализированный термин без пользы для SEO; пользователь, не знакомый с терминологией, не поймёт
- Теги «НГ (негорючие)», «0% асбеста», «Крупноформатные» — первый тег хорош, остальные слабо привязаны к ключевым запросам
- CTA «Оставить заявку» — нет конкретики, что именно получит пользователь
- **Критическое:** нет текстового блока в hero с упоминанием ключевых производственных характеристик (ГОСТ, К0) — информация прячется в блоке benefits
- Блок audience не содержит ключевых слов по типам объектов (жилые, коммерческие, социальные)

**Рекомендации:**
1. H1: «Фиброцементные плиты для навесных вентилируемых фасадов» (+НВФ)
2. Subtitle: «Негорючие (НГ, К0), без асбеста. Крупный формат до 3050 мм. Соответствие ГОСТ.»
3. Добавить в hero абзац с ключевыми словами: «Поставляем фиброцементные фасадные панели серий NATURE, POLISHED, TEXTURE, WALYPAN для жилых, коммерческих и социальных объектов»
4. CTA: «Получить техническое предложение» или «Запросить образцы»

---

### catalog.php — Каталог

**Целевые запросы:** купить фиброцементные панели, размеры фиброцементных плит, цвета фиброцементных панелей, NATURE POLISHED TEXTURE WALYPAN

**Проблемы:**
- **H1 полностью отсутствует** — страница начинается сразу с H2 продуктов. Это критическая SEO-ошибка
- Продуктовые H2 слабые: «Натуральные панели серия NATURE» — нет ключевого слова «фиброцементные»
- Описания продуктов эстетические («индустриальная эстетика»), но не включают технические преимущества и практическую пользу
- Блок «Характеристики» содержит технические данные без контекста применения
- Цветовые свотчи используют несогласованные имена (Белый/Бежевый светлый/Бежевый — не совпадают с блоком цветов в palette: Жемчужный/Бежевый)
- Изображения NATURE и POLISHED загружаются с tildacdn.com (внешний домен) — нет индексации

**Рекомендации:**
1. Добавить H1 в начало страницы: «Каталог фиброцементных панелей BLP Board»
2. Переписать H2: «Фиброцементные панели серии NATURE — фактура под бетон»
3. Добавить в каждое описание продукта: область применения + одну практическую характеристику
4. Перенести изображения с tildacdn.com в /blp/images-convert/

---

### architect.php — Архитекторам

**Целевые запросы:** фиброцементные панели для архитекторов, техническая документация фасадные панели, образцы фиброцементных плит

**Проблемы:**
- H1 «Кому это принесёт пользу» — полностью нерелевантный запросам заголовок, не содержит ни одного ключевого слова
- Три карточки (Архитекторам, Проектировщикам, Дизайнерам) перекрываются по смыслу; SEO-сигнал размыт
- Раздел «Партнерская модель взаимодействия» — слабый CTA «Оставить заявку» без конкретики
- Нет упоминания конкретных инструментов: BIM-модели, DWG-чертежи, паспорта изделий — это реальные запросы архитекторов
- Блок поддержки «Все, чтобы заложить материал в проект» — хороший заголовок, но список слишком краткий

**Рекомендации:**
1. H1 заменить на: «Фиброцементные панели BLP Board для архитекторов и проектировщиков»
2. Добавить конкретику в список поддержки: «DWG-узлы крепления», «BIM-объекты по запросу», «Паспорта изделий»
3. CTA в партнерской карточке: «Запросить техническую документацию»

---

### diler.php — Дилерам

**Целевые запросы:** стать дилером фиброцементных панелей, дилерская программа фасадные материалы, оптовые поставки фиброцементных плит

**Проблемы:**
- H1 «Дилерская программа по фиброцементным плитам» — хороший, но не содержит бренд и географию
- Hero subtitle «Долгосрочная работа через проектные продажи» — жаргон без расшифровки
- Hero CTA `href="#"` — ссылка на якорь не работает (должна вести на форму)
- Блок «Почему это интересно?» — слишком абстрактный; нет цифр, нет конкретных условий партнёрства
- Нет упоминания маржинальности, условий входа, требований к дилеру — то, что реально ищут потенциальные дилеры

**Рекомендации:**
1. Hero subtitle: «Продавайте фиброцементные панели для жилых, коммерческих и социальных объектов — мы обеспечиваем техподдержку и сопровождение сделки»
2. Исправить `href="#"` → `href="/blp/contacts#form"` или якорь на форму внизу страницы
3. Добавить в «Почему это интересно?» конкретику: «Решение по фасаду принимается по 4 критериям: эстетика, огнестойкость, логистика, цена — первые три мы закрываем полностью»

---

### devops.php — Застройщикам

**Целевые запросы:** фиброцементные панели для застройщиков, фасадные панели жилой комплекс, поставка фасадных плит ГОСТ

**Проблемы:**
- H1 хороший: «Фиброцементные панели для застройщиков»
- Hero subtitle содержит ключевую ценность, но CTA «Задать вопрос» — слабый
- `href="/contacts"` без `/blp/` — битая ссылка в локальной среде
- Блок «Юридическая и техническая надежность» — единственное место где упоминается ГОСТ; стоит вынести в более заметное место
- Нет упоминания объёмов поставок, сроков, складских возможностей — критично для застройщиков

**Рекомендации:**
1. CTA: «Запросить расчёт поставки» или «Получить коммерческое предложение»
2. Исправить `href="/contacts"` → `href="/blp/contacts"`
3. Добавить в hero список с конкретикой: «Поставки от X м² | Срок: X недель | Документы для госэкспертизы включены»

---

### contacts.php — Контакты

**Проблемы:**
- H1 «Контакты» — слишком короткий, не содержит бренд или гео
- Subtitle «Свяжитесь с нами удобным способом» — общее место
- Форма дублирует блок `contact-form.php` в конце страницы — две формы на одной странице без необходимости
- Нет упоминания времени работы
- Яндекс-карта подключена, но нет Schema.org LocalBusiness разметки (смотри задачу #8)

**Рекомендации:**
1. H1: «Контакты BLP Board — Одинцово, Московская область»
2. Subtitle: «Работаем с архитекторами, застройщиками и дилерами. Ответим в течение рабочего дня.»
3. Убрать дублирующий include `contact-form.php` в конце файла (строка 166)
4. Добавить часы работы в карточку контактов

- «Сколько стоят фиброцементные панели? Как получить цену?»

**Ключевые слова:** фиброцементные панели отличие, К0 что значит фасад, образцы фиброцементных панелей, фиброцемент цена за м2

## 8. Резюме критических ошибок (топ-5)

1. **catalog.php: H1 отсутствует** — страница полностью лишена H1, что является грубой SEO-ошибкой
2. **architect.php: H1 нерелевантный** — «Кому это принесёт пользу» не содержит ни одного целевого ключевого слова
3. **diler.php: CTA href="#"** — кнопка «Обсудить сотрудничество» ведёт в никуда, конверсия нулевая
4. **Все страницы: жаргон без расшифровки** — НГ, К0, НВФ используются без пояснений; аудитория вне отрасли не понимает
5. **Транзакционный интент не закрыт** — запросы «купить», «цены», «стоимость м²» не имеют посадочной страницы
