## 🏠 Страница 1: Главная (index.php)

### Текущие проблемы:
- Hero блок имеет `padding-top: 40px` → отступ от хедера
- Отступы от краёв не одинаковые (разные у разных секций)


- ✓ Размеры заголовков OK (h1: 24px, h2: 20px)?

## 🎯 Глобальные изменения (ВСЕ страницы)

### Удалить !important у шрифтов:
```css
/* ДО (неправильно) */
h1 { font-size: 42px !important; }

/* ПОСЛЕ (правильно) */
@media (max-width: 900px) {
  h1 { font-size: 28px; }
}

@media (max-width: 640px) {
  h1 { font-size: 24px; }
}
```

### Унифицировать breakpoints:
- **Все файлы должны использовать одну систему:** 1200px, 960px, 640px
- Убрать outliers: 480px, 768px, 900px, 1024px (они конкурируют)

### Единый горизонтальный отступ: **16px**
```css
@media (max-width: 640px) {
  main > * {
    padding-left: 16px !important;
    padding-right: 16px !important;
  }
}
```
