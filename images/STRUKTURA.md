# Структура папок изображений

```
images/
├── pages/
│   ├── index/           — изображения для главной страницы
│   ├── catalog/         — изображения каталога (серии, палитра)
│   ├── devops/          — изображения для застройщиков
│   ├── diler/           — изображения для дилеров
│   ├── kreplenie/       — изображения способов крепления
│   ├── projects/        — изображения проектов (объекты)
│   ├── sertificate/     — изображения сертификатов
│   ├── architect/       — изображения для архитекторов
│   ├── contacts/        — изображения контактов
│   └── policy/          — изображения политики
├── blocks/
│   ├── hero/            — изображения hero-блоков
│   ├── products/        — изображения продуктов
│   ├── benefits/        — изображения преимуществ
│   ├── partners/        — логотипы партнеров
│   ├── objects/         — изображения объектов
│   └── contact-form/    — изображения контактной формы
├── shared/
│   ├── header/          — изображения хедера (логотип, значки)
│   ├── footer/          — изображения футера
│   └── common/          — общие изображения (фавикон и т.д.)
└── STRUKTURA.md         — этот файл
```

## Как добавить изображения

1. Поместите изображение в соответствующую папку
2. Используйте путь `/blp/images/[category]/[page]/[filename]` в PHP файлах
3. Замените все ссылки на `https://tildacdn.com/...` на новые пути

**Пример замены:**
```
Было:  src="https://static.tildacdn.com/tild6435.../17.png"
Стало: src="/blp/images/pages/projects/17.png"
```
